# 앱인토스 검수 메모

작성일: 2026-07-11

## 서비스 성격

운명연구소는 개인 개발자가 운영하는 무료 비게임 미니앱입니다. 사주, 타로, 신점 메시지, 별자리, 오늘의 운세, 이주의 운세, 이달의 운세를 제공합니다.

## 비용/광고/로그인

- 모든 기능은 무료입니다.
- 광고가 없습니다.
- 결제 기능이 없습니다.
- 토스페이 또는 외부 결제 유도가 없습니다.
- 실제 토스 로그인 `appLogin()`을 호출하지 않습니다.
- `getAnonymousKey()`는 내부 사용자 식별과 저장 데이터 구분에만 사용합니다.
- 이메일 및 비밀번호 회원가입이 없습니다.
- 별도 앱 권한 요청이 없습니다.

## 데이터 처리

- 사주 입력값은 앱 안에서 계산에 사용합니다.
- 서버 데이터베이스를 사용하지 않습니다.
- 사용자가 저장을 선택한 결과만 앱인토스 Storage 또는 기기 localStorage fallback에 저장합니다.
- 저장 데이터는 하단 `내 기록` 또는 설정 화면에서 삭제할 수 있습니다.
- 공유 메시지에는 이름, 생년월일, 출생시간, 성별, anonymous hash를 포함하지 않습니다.

## 기능 설명

- 사주: 생년월일시를 바탕으로 원국, 오행, 십성, 대운, 세운, 신살, 인연 흐름을 보여줍니다.
- 오늘·이주·이달 운세: 같은 입력과 같은 날짜 기준에서 일관된 점수와 참고 메시지를 제공합니다.
- 별자리: 별자리를 직접 선택하거나 생일 기준으로 자동 계산해 오늘/주간/월간 흐름을 보여줍니다.
- 타로: 사용자가 펼쳐진 카드 덱에서 직접 3장을 선택하면 현재, 숨은 흐름, 조언을 제공합니다.
- 신점 메시지: 생년월일 없이 주제를 선택해 자동화된 참고 메시지를 확인합니다.

## 면책 안내

모든 결과는 전통 명리 및 카드·메시지 해석을 바탕으로 한 오락·자기이해 목적의 참고 콘텐츠입니다. 의료, 법률, 투자, 대출, 도박, 취업, 결혼 등 중요한 결정을 대신하지 않습니다. 신점 메시지는 실제 무속인의 상담, 신적 계시, 영적 사실 확인이 아닙니다.
