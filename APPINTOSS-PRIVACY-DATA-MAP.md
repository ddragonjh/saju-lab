## 처리 원칙

- 서버 데이터베이스를 사용하지 않는다.
- 사용자가 저장을 선택하지 않아도 모든 기능은 동작한다.
- 공유, URL, 콘솔 로그에 이름/생년월일/출생시간/성별/anonymous hash를 포함하지 않는다.
- 실제 토스 로그인, 토스 프로필 조회, 토큰 저장은 하지 않는다.

## 외부 SDK

- 앱인토스 Web Framework: `getAnonymousKey`, `Storage`, back event, share link 사용
- Toss Design System: UI 컴포넌트/스타일 기반 사용
- Google Fonts: 앱인토스 빌드에서는 사용하지 않고 시스템 폰트를 우선 사용

## 이용자 권리와 삭제

사용자는 하단 `내 기록`에서 항목별 결과를 삭제하거나, 설정 화면에서 앱 데이터 전체를 초기화할 수 있다. 서버 저장소가 없으므로 운영자가 원격으로 사용자의 기기 저장 데이터를 조회하거나 삭제할 수 없다.
