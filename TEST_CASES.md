# 운명연구소 테스트 케이스

## 계산

- 23시 기준과 0시 기준의 일주 날짜 경계가 달라지는지 확인
- 0시 출생 입력이 정상 계산되는지 확인
- 입춘 전후의 년주 변경 확인
- 월주 변경 절기 전후 확인
- 윤년 2월 29일 입력 확인
- 태어난 시간 모름 선택 시 시주 미상 처리 확인
- 잘못된 날짜 입력 시 오류 표시 확인
- 1930년과 2035년 경계 입력 확인
- 해외 출생자는 정밀 진태양시 미지원 고지가 노출되는지 확인

## 개인정보와 보안

- localStorage 삭제 버튼 실행 후 로컬 계정, 세션, 오늘 확인 기록 삭제 확인
- 새 브라우저 또는 새 프로필에서 기존 로컬 계정이 존재하지 않는지 확인
- 프리미엄 코드를 등록하지 않은 상태에서 프리미엄 결과가 열리지 않는지 확인
- XSS 문자열(`<img src=x onerror=alert(1)>`)을 이름에 입력해도 HTML로 실행되지 않는지 확인
- URL 파라미터에 이름·생년월일이 붙지 않는지 확인

## 접근성

- 키보드만으로 사주 입력부터 결과 확인까지 이동
- 회원 모달에서 Tab 포커스가 모달 안에 머무르는지 확인
- ESC로 회원 모달이 닫히는지 확인
- 오류와 결과 영역이 `aria-live`로 갱신되는지 확인
- 모바일 320px 너비에서 푸터 법적 링크, 사업자 정보, 환불 안내가 보이는지 확인

## 만세력 기준값

현재 엔진은 경량 천문 근사식을 사용합니다. 독립 만세력과의 대조표는 운영자가 신뢰할 수 있는 기준 출처를 선택한 뒤 이 문서에 출처와 함께 추가해야 합니다. 검증 범위로 표시하는 연도는 1930년부터 2035년까지입니다.
