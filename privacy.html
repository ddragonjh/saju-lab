2. 저장 위치와 보유 기간

저장되는 정보는 서버가 아니라 이용자의 브라우저 localStorage에 저장됩니다. 운영자는 이용자의 브라우저 저장소에 접근할 수 없습니다.

보유 기간은 이용자가 브라우저 사이트 데이터를 삭제하거나, 아래 “저장 정보 삭제” 기능을 실행할 때까지입니다.

3. 삭제 및 탈퇴 방법

현재 계정은 실제 서버 회원가입이 아니라 이 기기에만 저장되는 로컬 계정입니다. 탈퇴는 저장 정보 삭제와 동일합니다.

  • 사이트 내 버튼:
  • 브라우저 설정: 사이트 데이터 또는 localStorage 삭제
  • 공용 기기에서는 이용 후 로그아웃과 브라우저 데이터 삭제를 권장합니다.

4. 제3자 제공과 외부 리소스

이용자가 입력한 생년월일, 이름, 이메일, 프리미엄 코드 입력값을 제3자에게 제공하지 않습니다.

Google Fonts 외부 호출은 제거했습니다. 현재 페이지는 같은 도메인의 정적 파일, 이미지, CSS, JS를 사용합니다. 이메일 문의 링크를 누르는 경우 이용자의 이메일 앱이 열릴 수 있습니다.
