# 앱인토스 이전 감사 기록

작성일: 2026-07-11  
대상: 운명연구소 앱인토스 무료 비게임 미니앱

## 이전 원칙

- 기존 GitHub Pages 웹사이트는 수정 없이 계속 운영한다.
- 앱인토스 코드는 `appintoss/`에 별도 구성한다.
- 기존 계산 엔진과 해석 데이터는 `shared/` TypeScript 모듈로 옮겨 재사용한다.
- 무료 기능은 축소하지 않고 앱 화면에 맞게 재구성한다.
- 유료, 결제, 광고, 로컬 회원가입, 실제 토스 로그인은 출시 빌드에서 제거한다.

상세 기능 표는 `CURRENT-FEATURE-INVENTORY.md`에 기록했다.

## 앱인토스 SDK 적용

- `getAnonymousKey()`는 비게임 미니앱에서 anonymous hash를 받아 내부 사용자 식별에 사용한다.
- `Storage`는 사용자가 저장을 선택한 결과와 최근 기록을 저장한다.
- `graniteEvent.addEventListener('backEvent')`로 앱 뒤로가기 이벤트를 처리한다.
- `getTossShareLink()`와 `share()`를 사용할 수 있을 때 앱 딥링크 공유를 시도하고, 실패 시 Web Share API/클립보드로 fallback한다.

## 개인정보/보안 변경

- 이메일, 비밀번호, 세션 저장 제거
- URL 쿼리에 이름/생년월일/출생시간 저장 금지
- 콘솔 로그에 개인정보 출력 금지
- 사용자 입력을 HTML 문자열로 삽입하지 않음
- 저장 데이터 전체 삭제와 항목 삭제 제공
- 공유 메시지에 이름, 생년월일, 출생시간, 성별, anonymous hash 제외
