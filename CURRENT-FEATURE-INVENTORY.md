## 앱인토스 무료 앱에 포함할 무료화 범위

웹에서는 로컬 계정 뒤에 잠겨 있던 결혼 후보 연도, 학업, 인생 4막, 건강 참고, 대운, 세운, 신살을 앱에서는 모두 무료로 연다. 반면 `js/premium.js`와 `js/premium-hashes.js`에 있는 이용권, 결제 고지, 궁합, 프리미엄 질문, 타이밍 캘린더는 앱 빌드에 포함하지 않는다.

## 저장 데이터 기준

- 웹 현재 상태: localStorage에 로컬 계정, 세션, 프리미엄 상태, 오늘의 운세 확인 여부가 저장될 수 있다.
- 앱 이전 기준: 앱인토스 `getAnonymousKey()` hash 또는 게스트 UUID로 네임스페이스를 나누고, 사용자가 저장을 선택한 결과만 저장한다.
- URL, 콘솔 로그, 공유 텍스트에는 이름, 생년월일, 출생시간, 성별, anonymous hash를 포함하지 않는다.

## 외부 통신 기준

- 기존 웹 계산/리딩 기능은 외부 API 호출 없이 정적으로 동작한다.
- 앱은 앱인토스 SDK 호출(`getAnonymousKey`, `Storage`, back event) 외 서버 통신 없이 동작하게 한다.
- 토스 로그인 `appLogin`, 토큰 교환 API, 결제 SDK, 광고 SDK는 현재 출시 빌드에서 금지한다.
