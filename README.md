# 운명연구소 — 사주팔자 심층 해석 사이트

프리미엄 다크+골드 디자인의 사주 해석 웹사이트. 서버 없이 동작하는 순수 정적 사이트로,
만세력 계산(절기 기준 년·월주, 율리우스일 기준 일주, 시주)부터 십성·지장간·대운·세운·신살·
결혼 후보 연도까지 브라우저에서 직접 계산합니다.

## 실행
- 그냥 `index.html`을 더블클릭해도 동작합니다 (완전 정적).
- 개발 미리보기: `npx http-server . -p 8321`

## 배포 (무료, 5분)
1. [Netlify Drop](https://app.netlify.com/drop) 접속 → 이 폴더를 통째로 드래그 → 즉시 URL 발급
2. 또는 GitHub Pages / Vercel — 빌드 설정 없이 폴더 그대로 업로드
3. 발급된 URL을 `shorts/00-…가이드.md`의 프로필 링크 자리에 넣기

## 구조
```
index.html        — 페이지 (히어로/입력폼/결과/원리/FAQ)
css/style.css     — 딥네이비+골드 프리미엄 테마
js/engine.js      — 만세력·명리 계산 엔진
js/texts.js       — 해석 텍스트 DB (일간 10종, 오행, 십성, 배우자궁, 신살…)
js/app.js         — 폼·렌더링 컨트롤러
shorts/           — 숏츠 채널 런칭 패키지 (브랜딩·대본 10편·캘린더)
```

## 계산 방식 · 한계
- 절기 날짜는 천문 근사식(1930~2025 검증)으로 계산 — 절입 '당일' 출생은 시각 단위 경계에 걸릴 수
  있어 결과 화면에 경계 경고를 표시합니다.
- 진태양시 보정(−30분)은 기본 켜짐, 23시 이후 출생은 다음 날 일주로 계산(자시 기준).
- 음력 입력은 미지원 — 잘못된 변환보다 안내가 낫다고 판단, 양력 변환 후 입력을 안내합니다.
- 본 해석은 명리학 기반 참고용 콘텐츠입니다 (사이트 푸터에 고지 포함).
