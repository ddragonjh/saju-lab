# 숏츠 팩토리 — 영상 완전 자동 생성기

대본 → 신경망 TTS 내레이션 → 장면별 배경 전환 영상 → 업로드용 MP4까지 전부 자동.
**무료 · 무제한 · 워터마크 없음.**

## 사용법 (터미널에서)
```bash
cd Desktop\사주\shorts\factory
node render.js ep01     # 1편만
node render.js all      # 10편 전부 (약 25분)
```
완성본은 `shorts/output/` 폴더에 `ep01_결혼….mp4` 형태로 저장됩니다.

## 스펙
- 1080×1920 (9:16), 30fps, H.264+AAC — 유튜브 쇼츠/릴스/틱톡 바로 업로드 가능
- 내레이션: Microsoft Edge 신경망 음성 `ko-KR-SunHiNeural` (자연스러운 한국어)
- 장면 길이 = 내레이션 길이에 자동 동기화 (편당 35~45초)
- 배경 6종 자동 순환: 달밤 → 붉은 황혼 → 12지신 링 → 촛불 → 수묵 산세 → 한자 기운
  - 켄번즈 줌·별 반짝임·불꽃 흔들림 등 모든 장면이 항상 움직임
- 마지막 장면: "내 사주 10초 무료 해석 / 프로필 링크 클릭" CTA 고정

## 새 에피소드 추가
`episodes.js`에 항목 하나 복사해서 추가하면 끝:
```js
{ id:'ep11', title:'제목',
  scenes:[ {text:'화면에 보일 자막\n(줄바꿈은 \\n)', narr:'실제로 읽어줄 내레이션 문장'}, … ] }
```
- 장면 4~6개, narr는 한 문장씩 짧게 (한 장면 5~8초가 최적)
- 목소리 바꾸기: render.js에서 `ko-KR-SunHiNeural`을 `ko-KR-InJoonNeural`(남성)으로
