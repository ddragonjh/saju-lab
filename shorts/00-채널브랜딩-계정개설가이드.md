## 2. 계정 개설 순서 (사람이 하는 30분)

### ① 구글/유튜브 (10분)
1. youtube.com → 로그인 → 프로필 → "채널 만들기"
2. 채널명 `운명연구소`, 핸들 `@unmyeong_lab`
3. 채널 아트/프로필 업로드 → 정보란에 한 줄 소개 + 사이트 링크
4. 설정 → 채널 → "어린이용 아님" 선택 (사주 콘텐츠는 일반)

### ② 인스타그램 (10분)
1. 앱에서 신규 계정 → 사용자명 `unmyeong_lab`
2. **프로페셔널 계정(크리에이터)** 전환 — 카테고리 "디지털 크리에이터"
3. 바이오에 한 줄 소개 + 사이트 링크, 하이라이트 커버 골드 톤 통일

### ③ 틱톡 (10분)
1. 앱 가입 → 아이디 `unmyeong_lab`
2. 프로필 편집 → 소개 + (팔로워 1천 이상부터 바이오 링크 가능하므로 초기엔 "유튜브: 운명연구소" 표기)

> ⚠ 3개 모두 **같은 이메일 + 같은 프로필 이미지**로 만들어야 브랜드 검색이 잡힙니다.
> ⚠ 개설 첫 주에 하루 3개 이상 올리면 스팸 분류될 수 있음 → 캘린더(02 파일)의 주 3회 페이스 준수.

## 3. 영상 제작 파이프라인 (편당 15분)

1. `01-숏츠대본-10편.md`에서 대본 1편 선택
2. **CapCut(무료)** → 9:16 새 프로젝트 → 검은 배경 템플릿
3. 자막: 대본의 [자막] 그대로 붙여넣기 (폰트: 눈누 "지마켓산스" 또는 CapCut 명조 계열, 골드 컬러)
4. 내레이션: CapCut "텍스트 음성 변환(TTS)" — 차분한 남/여 보이스 or 직접 녹음
5. 배경: 무료 소스(Pexels "night sky", "candle", "ink") + 잔잔한 국악/로파이 BGM (CapCut 상업용 무료 라이브러리)
6. 마지막 3초: 고정 CTA 화면 (사이트 링크 + "무료 사주 해석")
7. 같은 영상을 유튜브 쇼츠/릴스/틱톡에 **동시 업로드** (해시태그만 02 파일대로 교체)

## 4. 절대 금지 (채널 계정 보호)

- "수익 보장·합격 보장·결혼 보장" 등 확정 표현 금지 → 반드시 "~하는 흐름/기운입니다"
- 특정인 실명 사주 풀이 금지 (연예인 포함 — 명예훼손 리스크)
- 공포 마케팅 금지 ("이 사주는 망합니다" ❌ → "이 기운은 이렇게 다스립니다" ⭕)
- 모든 영상 설명란에 고지: "본 콘텐츠는 명리학 기반 오락·참고용입니다."
