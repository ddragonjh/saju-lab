# 운명연구소 — 4주 업로드 캘린더 & 플랫폼별 해시태그

## 1. 업로드 원칙
- **주 3회 (월·수·금), 오후 6~8시** 업로드 (퇴근·하교 시간대 = 운세 콘텐츠 골든타임)
- 같은 영상을 유튜브 쇼츠 → 인스타 릴스 → 틱톡 순으로 같은 날 올림 (해시태그만 교체)
- 댓글 첫 24시간은 최대한 답글 (초기 알고리즘의 핵심 신호)
- 댓글 답글 템플릿: "생년월일시가 있으면 더 정확해요. 프로필 링크에서 무료로 보실 수 있습니다 🙂"

## 3. 플랫폼별 해시태그 세트

**유튜브 쇼츠** (설명란 3~5개만)
`#사주 #운세 #사주풀이 #shorts` + 에피소드 태그 1개

**인스타 릴스** (15~20개)
`#사주 #사주팔자 #운세 #사주풀이 #오늘의운세 #사주스타그램 #운세스타그램 #명리학 #신년운세 #결혼운 #재물운 #연애운 #궁합 #무료사주 #사주상담 #운명 #팔자 #역학 #점스타그램` + 에피소드 태그

**틱톡** (4~6개)
`#사주 #운세 #fyp #추천 #사주풀이` + 에피소드 태그

## 4. 성과 측정 (주 1회, 일요일 10분)
- 각 플랫폼 조회수·평균 시청 지속률·프로필 클릭 수 기록
- 시청 지속률 70% 미만 에피소드 → 훅(첫 3초) 교체 후 재업로드 실험
- 사이트 방문자 수와 대조해 "어떤 에피소드가 방문을 만드는지" 확인 → 그 주제만 시리즈화

## 5. 다음 단계 로드맵
1. 팔로워 1,000명 → 틱톡 바이오 링크 개방, 인스타 스토리 링크 활용
2. 월 방문 1만 → 사이트에 "프리미엄 심층 리포트(PDF)" 유료 상품 (기존 결제 모듈 재사용 가능)
3. 구독자 요청 기반 "궁합 보기" 기능을 사이트에 추가 → 커플 단위 바이럴
